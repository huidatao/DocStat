Document Statistics

Team Member: Huida Tao, Yifei Shen

This program has following classes to support several functions:

•charStat.scala
calculate the number of times each character appears in the document.
•alphaDict.scalacreate an alphabetized dictionary of each word that appears in the document.
•WordStat.scala
calculate the number of times each word appears in the document.
•topThreeSeq.scala
identify the three word phases that appear the most often in the text file and show the top three phrases and the number of times they appear in the document.The calculated file is resource/doc.txt. If you want to change the name of file, change the filapath in code as well.
